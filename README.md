# Códigos de Resolução de Exercícios em C++

## Como Executar os Códigos

Para executar os códigos em C++ neste repositório, você precisa de um compilador C++ instalado, como o GCC (g++) ou Clang.

### Passos para Compilar e Executar:

1. **Instale um compilador C++** (se ainda não tiver):
   - No macOS, você pode instalar o GCC via Homebrew: `brew install gcc`
   - Ou use o Xcode Command Line Tools: `xcode-select --install`

2. **Compile o código**:
   - Abra o terminal e navegue até a pasta do arquivo .cpp.
   - Execute o comando: `g++ nome_do_arquivo.cpp -o nome_do_executavel`
     - Exemplo: `g++ hello.cpp -o hello`

3. **Execute o programa**:
   - Execute o comando: `./nome_do_executavel`
     - Exemplo: `./hello`

### Exemplo Completo:
```bash
g++ hello.cpp -o hello
./hello
```

Certifique-se de que o arquivo .cpp esteja na mesma pasta ou ajuste o caminho conforme necessário.
